Portfolio Summary

A Webflow platform for a recurring Silicon Valley event series — built around a relational CMS that links speakers, startups, and events in a single schema. Designed to be fully managed by a non-technical team, the site has scaled to 29 events, 40 speakers, and 8,150+ attendees with zero code changes.

CMS Architecture

The CMS is built around a relational event schema — each event links to multiple speaker and startup entries, allowing the site to display rich, connected data from a single source of truth.

Events — date, poster image, registration link, multi-ref to speakers and startups
Speakers — name, photo, LinkedIn, bio; referenced per event
Startups — founder name, LinkedIn; referenced per event
Flyers Archive — image-only collection, paginated, auto-populates the archive page

Adding a new monthly event requires one CMS entry — the homepage carousel, archive page, and counter stats all update automatically.

Event Flyers Archive

The Flyers page serves as a visual archive of every past event — a paginated CMS grid that scales indefinitely. Each flyer links to its full event detail page.
