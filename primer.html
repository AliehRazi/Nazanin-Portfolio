Overview

Exploring how far Webflow can go — by building a fintech landing page that mimics real product behavior.

Structured as a long-form conversion flow — moving from clarity to decision without overwhelming the user
Interactions used intentionally to guide attention, not decorate — including pricing toggles, scroll reveals, and motion cues
Built entirely without custom code — pushing Webflow's native capabilities to their edge

Brief

Design a fintech landing page that converts — not just visually, but structurally

The constraint: build a page a real product team could ship, using only Webflow.

Approach

Start with structure, then layer interactions

The page was mapped as a long-form flow — from value → features → pricing → action — before any visual styling.

Outcome

A landing page that behaves like a product

Interactions are used to guide attention and reduce friction across a long-scroll experience.

Design

Minimal input, maximum clarity

The visual direction is intentionally restrained — a light interface, strong typographic hierarchy, and product UI used as the primary visual proof.

Instead of relying on decoration, the design focuses on structure. Each section moves from broad value to specific features and pricing, guiding the user through a deliberate long-form flow.

Key Feature

Interactions that earn attention

Every animated element serves the layout — not the other way around. The pricing toggle reduces friction at the decision point, while scroll reveals pace the reading flow. Motion is used to guide attention, not decorate.

01 / Primary

Pricing toggle

Switches between monthly and yearly plans with smooth state transitions — allowing quick comparison without interrupting the decision flow.

02 / Scroll

Scroll reveals

Staggered content reveals control pacing across the page — preventing information overload in a long-form layout.

03 / Motion

Word marquees

Subtle, multi-directional motion reinforces key terms and creates rhythm without distracting from the content.

04 / Proof

Floating mockup hero

Layered mobile UI with depth and slight motion establishes product credibility before a single word is read.

05 / System

Feature card grid

Hover and scroll-based interactions turn static content into a browsable, scannable system.

06 / Responsive

Responsive breakpoints

Interactions adapt across screen sizes — maintaining clarity and consistency without layout breaks.

Pricing toggle interaction

A focused demo of the primary decision-point interaction.

What I Practiced

Webflow from first principles

This project was built to demonstrate mastery of the Webflow toolkit — not just the visual output, but the thinking behind structure, performance, and interaction design.

Webflow Interactions — scroll-triggered animations, toggle states, hover effects, all without custom code
Responsive layout system — breakpoints from desktop to mobile without any layout breaks or overflow
Typography hierarchy — headline, subheadline, body, and caption working together across a long-form page
Asset optimisation — .avif and .svg formats throughout for performance without visual compromise
SaaS page architecture — hero → proof → features → pricing → CTA as a deliberate conversion funnel
