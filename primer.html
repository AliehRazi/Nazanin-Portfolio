Portfolio Summary

A class project built to master Webflow's full interaction and layout toolkit — structured as a real fintech product landing page. The brief was simple: build something that converts. The result demonstrates command of SaaS page architecture, Webflow Interactions, responsive layouts, and typographic hierarchy.

Concept & Brief

Primer is a fictional fintech product — a digital payments and finance app — used as the vehicle for practicing real-world Webflow techniques. The design challenge was to build a landing page that feels production-ready, not like a student exercise.

The structure follows a proven SaaS conversion pattern: hero → social proof → features → pricing → CTA — executed with intentional visual restraint.

Hero section — floating mobile mockups with strong typographic lead

Interactions & Layout

The page showcases a range of Webflow-native interactions that demonstrate front-end thinking beyond static layouts:

Pricing toggle — monthly/yearly switch with animated state changes
Diagonal word marquees — multi-directional scrolling text rows using CSS animations
Feature card grid — image + text cards with hover interactions
Mobile app mockup hero — layered floating UI screenshots with depth
Section transitions — smooth scroll-triggered reveals throughout

Feature sections — mobile mockups with layered card compositions

What I Practiced

This project was specifically used to practice and demonstrate mastery of:

Webflow Interactions — scroll-triggered animations, toggle states, hover effects
Responsive design — breakpoints from desktop to mobile without layout breaks
Typography hierarchy — headline, subheadline, body, and caption at scale
Asset optimisation — .avif and .svg formats for performance
